<br/>
<p>For GUI ==> <i>Run</i> <b>src\gui_menu\menu\ClientMainMenu.java + src\gui_menu\menu\ServerMainMenu/java</b></p>
<p>or</p>
<p>For Console Mode(may not work in UDP) ==> Run <b>src\console_menu\Menu.java<p>
